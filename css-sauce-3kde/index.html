Hvad en selector?

Hvorfor ikke altid bruge tag-selectors?
Hvornår skal vi bruge IDs?
Hvornår skal vi bruge classes?

Tag-selectoren


  • Rammer alle af den slags elementer
  • Laveste prioritet af alle selectors

Class-selectoren


  • Kan bruges på uendeligt mange elementer
  • Sammen class, kan bruges på flere forskellige tags
  • Har højere prioritet end tag-selectoren

ID-selectoren


  • Skal være unik - må kun optræde en gang på siden
  • Højeste prioritet - tager prioritet over de to andre

Pseudo-selectors

  • :visited - besøgte links
  • :link - ikke-besøgte links
  • :hover - når musen holdes over elementet

CSS-syntaxen - Generelle regler


  • Alle blokke skal åbnes og lukkes med { og }
  • Farver opgives i RGB eller HEX (##11CCFF)
  • ALTING ER CASE-SENSATIVE

CSS-syntaxen - selectoren



p { }
.class { }
#id { }

Protip: CSS og HTML tage ikke hensyn til mellemrum og afsnit, så længe de ikke står inden i et tag

CSS-synaxen - properties


div {
    background-color: hotpink;
    background-image: url("adresse") (repeat|no-repeat|repeat-x|repeat-y); 
    
    text-align: center; 
    
    margin: 0 auto; /** ...med display block & width **/
    
    color: red;
    font-size: 32px;
    font-weight: bold; 
    font-style: italic; 
    text-decoration: underline; 
}

Opgave 1


Du skal lave den første tekstblok indholdende teksten "En stump tekst" rød.

Hvordan vil du skrive CSS-selectoren?
Ville du ændre noget i HTMLen?

Løsning 1 (Simpel)

HTML:

<p>En stump tekst</p>
til 
                                <p class="redText">En stump tekst</p>

CSS:

.redText{
    color: red;
}

Løsning 2 (Advanceret)


div:first-child p{
    color: red;
}

Opgave 2


Lav overskriften understeget og skift farven til f.eks. #bada55

Hvordan vil du skrive CSS-selectoren?
Ville du ændre noget i HTMLen?

Løsning 1 (Simpel)

HTML:

<h1>Overskrift goes here</h1>
til 
<h1 id="overskrift">Overskrift goes here</h1>

CSS:

#overskrift{
    color: #bada55;
    text-decoration: underline;
}

Løsning 2 (Advanceret)


div:first-child h1{
    color: #bada55;
    text-decoration: underline;
}
